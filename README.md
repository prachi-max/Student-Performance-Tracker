# About The Project

The AI Student Performance & Task Management System is an intelligent academic productivity platform designed to help students manage tasks, monitor academic performance, and improve productivity using Artificial Intelligence and Machine Learning.

The platform analyzes task completion patterns, academic activities, and productivity trends to generate personalized insights, subject recommendations, and AI-based suggestions for students. It helps users understand their weak areas, improve consistency, and optimize study performance through data-driven analysis.

The system also includes an AI Voice Assistant that provides smart dashboard updates, interactive support, reminders, and quick accessibility features.

This project combines Web Development, Machine Learning, Data Visualization, and AI Assistance into a single smart educational platform.

# Getting Started

To get a local copy up and running, follow these steps.

---

# Installation

Clone the repository:

```bash id="0m1vya"
git clone https://github.com/prachi-max/Student-Performance-Tracker.git
```

Move to the project directory:

```bash id="p9z4yr"
cd Student-Performance-Tracker
```

---

# Backend Setup

Move into backend folder:

```bash id="x5e7sn"
cd backend
```

Install backend dependencies:

```bash id="q0mz44"
npm install
```

Run backend server:

```bash id="g1z2m4"
node server.js
```

Or using nodemon:

```bash id="2v9e8f"
npx nodemon server.js
```

---

# Frontend Setup

Move into frontend folder:

```bash id="t9m0wx"
cd frontend
```

Install Angular dependencies:

```bash id="r6b3xy"
npm install
```

Run Angular application:

```bash id="v4n8pd"
ng serve
```

Open browser:

```bash id="j3e0ft"
http://localhost:4200
```

# Project Workflow
